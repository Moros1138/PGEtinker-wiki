## Drawing Primitive Shapes

### Points

Draws a point at (**x**,**y**) with the provided **color**.

```cpp
// with integers
Draw(5, 5, olc::WHITE);

// with the olc::vi2d
Draw(olc::vi2d{5, 5}, olc::WHITE);
```

**Color:** Color has a default value of ``olc::WHITE``
